# 经侦法规补充下载清单（需手动下载部分）

> 背景：典法查阅「经侦77类」专项已上线（V2.31.0），并把《公安机关管辖的刑事案件立案追诉标准的规定（二）》（公通字〔2022〕12号，77 条）作为离线主文档入库。本清单回答"还有哪些经侦相关法律法规/司法解释需要下载、但 AI 抓不全"的问题。
>
> 铁律：法条必须最新版、来源官方、逐字保真；软件离线内网使用，不依赖互联网。
> 实测结论：最高人民法院/最高人民检察院公报、国家法律法规数据库的法条正文多为 **PDF 或长文**，而抓取工具（WebFetch）对这类内容**会改写/摘要、无法保证逐字保真**，伪造货币（二）等少数干净短 HTML 才能稳定抓到全文。因此：能稳定抓全的已自动入库；凡涉及保真的，统一交你从官方源下载原文放入，最稳妥。

## 四、放入与关联方式（你下载后告诉我即可）

1. **司法解释**：保存为 UTF-8 `.txt`，命名「最高人民法院、最高人民检察院关于…….txt」，放入 `public/laws/司法解释/`；
   在 `public/laws/manifest.json` 的 `laws` 数组追加一条（`category:"司法解释"`），并把 `categories` 里司法解释 `count` 与顶层 `totalLaws` 各 +1。
2. **基础法律**：放入对应分类目录（如 `public/laws/行政法律/`），同上更新 manifest。
3. **专项关联**：若要在「经侦77类」专项详情页「关联司法解释」卡片出现，告诉我它对应**规定（二）第几条**，我更新 `src/pages/LegalKnowledge.tsx` 的 `EC_RELATED` 映射即可。
4. **我代劳**：你也可以直接把官方原文文本/PDF 发我，由我负责入库 + 关联 + 升版（保真以你给的官方文本为准）。
